Go Movies


Go Movies is a Movie review website where users can log in and keep track of movies they want to see, rate movies, and read movie reviews. It also lets critics log in and write movie reviews which can be viewed by other critics and users. It works in the same way for TV Shows as well. It uses the tmdb API service to populate data for movies, TV Shows and celebrities. Any user who hasn't logged in can also view and search for content but will not be allowed to do the aformentioned activities.
